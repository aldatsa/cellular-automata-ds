Cellular Automata DS

Etxean garatutako Nintendo DSrako automata zelular simulatzailea.

Deskargatu NintendoDSrako ROMa (v0.11.0) Ikusi GitLab-en

Pantaila-argazkiak

Cellular Automata DS eraikitzeko behar direnak

  • devkitPro
  • libnds

Jarraitu argibide hauek tresnak eta liburutegiak instalatzeko (nds-dev taldea).

Nola eraiki eta exekutatu Cellular Automata DS

Klonatu biltegi hau:

git clone https://gitlab.com/aldatsa/cellular-automata-ds.git

Sartu proiektuaren direktoriora:

cd cellular-automata-ds

eta exekutatu ondorengoa:

make

Konpilatzen amaitzen duenean, kopiatu sortu den cellularautomatads.nds fitxategia zure SD txartelaren errora.
